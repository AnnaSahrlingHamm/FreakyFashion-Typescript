# Freaky Fashion TypeScript

A responsive e-commerce application developed during the JavaScript 3 course in my Frontend Developer education.

## About

Freaky Fashion TypeScript is an e-commerce application built using TypeScript and modern frontend development principles. The project focuses on building a maintainable, strongly typed application with reusable components, dynamic routing, search functionality, and product administration.

This project allowed me to deepen my understanding of TypeScript and apply static typing to a larger frontend application.

## Features

* Dynamic product listing
* Product detail pages based on URL slugs
* Product search functionality
* Search results page
* Responsive design
* Product administration
* Add new products
* Database-driven content
* Reusable UI components

## Built With

* TypeScript
* Angular
* HTML5
* CSS3
* REST API

## What I Learned

During this project I practiced:

* TypeScript fundamentals
* Interfaces and type definitions
* Strong typing across components and services
* Angular component architecture

* Working with APIs
* Responsive web development
* Creating reusable UI components
